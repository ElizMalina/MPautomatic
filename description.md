# Описание программы

### Программа состоит из двух основных компонентов:

МП автомат, который проверяет, соответствует ли данная цепочка заданному формату.

process() - основная программа, которая ищет все подходящие цепочки во входной строке.

### Работа подпрограммы:
MPautomatic реализует МП автомат с семью состояниями: q0,q1,q2,q3,q4,f.


Алгоритм работы:

1) Начинает с состояния q0. Буффера Z0.

Выполняет переходы между состояниями согласно правилам:

1) Из q0 при символе 'x' и буффере Z0 переходит в q1 и кладет в буффер 'X'.
