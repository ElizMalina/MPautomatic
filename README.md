# MP automatic


:mortar_board: Проект выполнен в рамках курса "Теория автоматов и формальных языков" на 2 курсе СПБГУ

## :point_down: Задача: 
Построить порождающую грамматику для КС языка { $$x^{3n}b^nz^ma^{3m} | n,m \geq 0$$ }. Разработать распознающий МП автомат для цепочек данного КС языка на алфавитом {x,b,z,a} и реализовать его в виде подпрограммы. Разработать программу, распознающую цепочки этого языка на основе программной реализации МП автомата 

## :pencil2: Порождающая грамматика:
G=(N,T,P,S) , где 
  1) T={x,b,z,a}
  2) S - начальный символ
  3) N={S,A,B}, где

  A = $$x^{3n}b^n$$ 

  B = $$z^ma^{3m}$$

  4) P - правило вывода:

  S -> AB
     
  A -> $$e_A$$ | xxAb

  B -> $$e_B$$ | zBaaa


## :date: MП автомат:

P = ({ $$q_0,q_1,q_2,q_3,q_4,f$$ },{x,b,z,a},{X,B,Z,A,C, $$Z_0 $$ },δ, $$q_0,Z_0$$ ,{f})

при n,m=0:
1) $$δ(q_0,1,Z_0)$$ = { $$(q_1,CZ_0)$$ }
2) $$δ(q_0,e,Z_0)$$ = { $$(f,e)$$ }
3) $$δ(q_1,1,C)$$ = { $$(q_0,e)$$ }
4) $$δ(q_3,1,Z_0)$$ = { $$(q_1,CZ_0)$$ }
5) $$δ(q_4,1,Z_0)$$ = { $$(q_1,CZ_0)$$ }

в других случаях:
для $$x^{3n}b^n$$ 

6) $$δ(q_0,x,Z_0)$$ = { $$(q_1,XZ_0)$$ }
7) $$δ(q_1,x,X)$$ = { $$(q_2,e)$$ }
8) $$δ(q_2,x,Z_0)$$ = { $$(q_3,B)$$ }
9) $$δ(q_2,x,B)$$ = { $$(q_3,B)$$ }
10) $$δ(q_3,b,B)$$ = { $$(q_4,e)$$ }
11) $$δ(q_3,x,B)$$ = { $$(q_1,XB)$$ }
12) $$δ(q_4,b,B)$$ = { $$(q_3,e)$$ }

для $$z^ma^{3m}$$

13) $$δ(q_0,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
14) $$δ(q_3,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
15) $$δ(q_4,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
16) $$δ(q_1,z,Z)$$ = { $$(q_1,ZZ)$$ }
17) $$δ(q_1,a,Z)$$ = { $$(q_2,AZ)$$ }
18) $$δ(q_2,a,A)$$ = { $$(q_3,e)$$ }
19) $$δ(q_3,a,Z)$$ = { $$(q_1,e)$$ }
20) $$δ(q_3,e,Z_0)$$ = { $$(f,e)$$ }
21) $$δ(q_1,e,Z_0)$$ = { $$(f,e)$$ }
