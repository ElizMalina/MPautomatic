# MP automatic


:mortar_board: Проект выполнен в рамках курса "Теория автоматов и формальных языков" на 2 курсе СПБГУ

## :point_down: Задача: 
Построить порождающую грамматику для КС языка { $$x^{3n}b^nz^ma^{3m} | n,m \geq 0$$ }. Разработать распознающий МП автомат для цепочек данного КС языка на алфавитом {x,b,z,a} и реализовать его в виде подпрограммы. Разработать программу, распознающую цепочки этого языка на основе программной реализации МП автомата 

## :pencil2: Порождающая грамматика:
G=(N,T,P,S) , где 
  1) T={x,b,z,a}
  2) S - начальный символ
  3) N={S,A,B} 
  4) P - правило вывода:

  S => AB
     
  A => ε | xxAb

  B => ε | zBaaa

  Пример:
  ~~~
xxxbzaaa=xxxεbzεaaa=>xxxAbzBaaa=>AB=>S
  ~~~


## :date: MП автомат:

P = ({ $$q_0,q_1,q_2,q_3,q_4,f$$ },{x,b,z,a},{X,B,Z,A, $$Z_0 $$ },δ, $$q_0,Z_0$$ ,{f})

при n,m=0:
1) $$δ(q_0,ε,Z_0)$$ = { $$(f,ε)$$ }
2) $$δ(q_1,ε,Z_0)$$ = { $$(f,ε)$$ }
3) $$δ(q_4,ε,Z_0)$$ = { $$(f,ε)$$ }

в других случаях:
для $$x^{3n}b^n$$ 

4) $$δ(q_0,x,Z_0)$$ = { $$(q_1,XZ_0)$$ }
5) $$δ(q_1,x,X)$$ = { $$(q_2,ε)$$ }
6) $$δ(q_2,x,Z_0)$$ = { $$(q_3,BZ_0)$$ }
7) $$δ(q_2,x,B)$$ = { $$(q_3,BB)$$ }
8) $$δ(q_3,b,B)$$ = { $$(q_4,ε)$$ }
9) $$δ(q_3,x,B)$$ = { $$(q_1,XB)$$ }
10) $$δ(q_4,b,B)$$ = { $$(q_3,ε)$$ }

для $$z^ma^{3m}$$

11) $$δ(q_0,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
12) $$δ(q_3,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
13) $$δ(q_4,z,Z_0)$$ = { $$(q_1,ZZ_0)$$ }
14) $$δ(q_1,z,Z)$$ = { $$(q_1,ZZ)$$ }
15) $$δ(q_1,a,Z)$$ = { $$(q_2,AZ)$$ }
16) $$δ(q_2,a,A)$$ = { $$(q_3,ε)$$ }
17) $$δ(q_3,a,Z)$$ = { $$(q_1,ε)$$ }
